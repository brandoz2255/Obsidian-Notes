# Harvis Brain Index

## What This Is

This is the shared knowledge layer between Dulc3 and Harvis. Everything we learn, decide, or build together gets documented here.

## Structure

### Sessions
Daily conversation summaries. Technical decisions, problem-solving, and context from our work.
- [[Sessions/2026-05-05]]

### Learnings
Things Harvis learned from Dulc3. User preferences, corrections, and patterns.
- How Dulc3 works

- Tool preferences

### Decisions
Technical decisions we made together. Architecture choices, tool selections, and reasoning.
- Why we chose X over Y
- Infrastructure decisions
- Project direction

### Playbooks
Reusable workflows and procedures we've built.
- Essay writing workflow
- K8s deployment patterns
- Git workflows

## How It Works

1. **You write** in your notes using `[[wikilinks]]`
2. **I read** your recent notes every session
3. **I write back** session summaries and learnings here
4. **You reference** my notes in yours
5. **Knowledge compounds** both directions
